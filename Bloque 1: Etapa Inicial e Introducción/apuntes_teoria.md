## 🐍 5. El Lenguaje Python
Python es uno de los lenguajes de programación más populares y utilizados en el mundo actual (desarrollo web, inteligencia artificial, ciencia de datos, automatización). Sin embargo, tiene una particularidad en su aprendizaje:

* **El mito de la facilidad:** Aunque su sintaxis es limpia, legible y cercana al lenguaje humano, **aprender a programar en Python NO es fácil**. 
* **Pensamiento Programático:** El verdadero desafío no es memorizar los comandos, sino aprender a **pensar lógicamente**. Hay que estructurar el pensamiento de forma sistémica para resolver problemas antes de pasarlos a código.
* **Paradigma:** Es un lenguaje multiparadigma (soporta programación estructurada, orientada a objetos, funcional) y de alto nivel, lo que significa que abstrae la complejidad del hardware de la computadora.

## ⚙️ 7. Instalación y Funcionamiento de Python
Para que la computadora pueda entender y ejecutar nuestros archivos de código (`.py`), necesita un intermediario:

* **El Intérprete:** Python es un lenguaje **interpretado**. Esto significa que hay un programa (el intérprete) que lee nuestro código línea por línea, en tiempo real, y lo traduce a instrucciones que el procesador de la PC pueda ejecutar.
* **Instalación:** Al instalar Python en el sistema operativo, se descarga este intérprete y se configuran las variables de entorno (PATH) para que podamos invocarlo desde cualquier terminal usando el comando `python`.
* **Verificación:** La instalación correcta se comprueba desde la consola ejecutando el comando:
  ```bash
  python --version
