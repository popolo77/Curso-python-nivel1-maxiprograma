## 🎯 Objetivos de la Formación

* **Pensamiento Sistémico:** Comprender la lógica elemental de la programación y la estructura de los algoritmos antes de escribir sintaxis.
* **Resolución de Problemas:** Desarrollar programas eficientes y óptimos para problemáticas determinadas de forma secuencial e iterativa.
* **Lenguaje Técnico:** Adquirir vocabulario propio del ámbito de desarrollo de software (paradigmas, intérpretes, variables de entorno, etc.).
* **Consistencia:** Aplicar lo aprendido mediante la resolución práctica de más de 90 videos y desafíos planteados en el campus virtual.

## 📂 Estructura del Repositorio

El contenido se encuentra organizado de forma progresiva respetando las unidades oficiales del programa:

* `1-intro/`: Conceptos iniciales de la disciplina, configuración del entorno, resúmenes teóricos y primer algoritmo.
* `2-secuenciales/`: Operaciones matemáticas básicas, almacenamiento en memoria y flujo secuencial de datos.
* `3-condicionales/`: Toma de decisiones y lógica condicional mediante operadores relacionales (`if`, `elif`, `else`).
* `4-repetitivas/`: Estructuras de control iterativas, control de bucles por contador o por estado (`while`, `for`).
* `5-listas/`: Manejo de colecciones, vectores, arreglos y estructuración de datos indexados de igual tipo.
* `6-funciones/`: Modularización, pasaje de parámetros por valor/referencia y reutilización de código limpio.
* `7-archivos/`: Persistencia de datos local, lectura, escritura y manipulación de archivos externos.
* `8-recursividad/`: Conceptos de funciones autollamadas, casos base y lógica algorítmica avanzada.

## 🔌 Entorno de Desarrollo de la Unidad 1 (Punto 9)

Para trabajar de manera eficiente y profesional con Python en Visual Studio Code, se configuró el entorno utilizando las siguientes extensiones esenciales:

### 1. Python (Microsoft)
Es el motor principal y obligatorio. Habilita el soporte para autocompletado inteligente (*IntelliSense* con Pylance), detección de errores de sintaxis en tiempo real, formateo básico y herramientas avanzadas para la depuración (*debugging*) paso a paso de los scripts `.py`.

### 2. Black Formatter (Microsoft)
Un formateador de código estricto que automatiza el cumplimiento del estándar profesional **PEP 8** (la guía de estilos oficial de Python) cada vez que se guarda un archivo. Al organizar automáticamente los bloques, previene los errores humanos de indentación (espaciado obligatorio en Python).

### 3. Error Lens
Mejora drásticamente la visibilidad de los errores durante el aprendizaje. En lugar de exigir que el cursor pase por encima de una línea subrayada en rojo, resalta la fila completa en tiempo real y escribe el mensaje exacto de error directamente en el editor.

### 4. Material Icon Theme
Personaliza y moderniza la interfaz visual del explorador de archivos en VS Code. Asigna iconos específicos y fácilmente reconocibles a cada extensión de archivo, facilitando la navegación rápida (como el logo oficial de Python junto a cada script desarrollado).
